本程序采用python的django框架编写，该框架按照MVC模式设计，很好地契合了BS模型的要求。前端使用bootstrap框架与javascript，有种小清新风格。

使用方法：
1.安装python2.X,安装django
2.安装完毕后，在此文件路径下运行cmd命令: python manage.py runserver，然后看到服务器运行起来的提示
3.打开浏览器（最好是chrome或者firefox），输入网址：127.0.0.1/index，便可看到登录界面
